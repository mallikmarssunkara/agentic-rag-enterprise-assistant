# Screenshot Guide

Capture and save the following screenshots for submission evidence:

## Recommended Files

- `home_page.png`
  Capture the Streamlit landing screen with the sidebar visible.

- `upload_success.png`
  Capture the successful ingestion message after uploading sample documents.

- `question_answer.png`
  Capture a sample question together with the generated answer.

- `source_chunks.png`
  Capture the evidence section showing retrieved source chunks and scores.

- `test_results.png`
  Capture the terminal output showing successful `pytest` execution.

## Capture Tips

- Keep the full application window visible.
- Ensure file names and answer text are readable.
- Avoid including private paths, secrets, or unrelated desktop content.
- Use the provided public sample documents when generating screenshots.
