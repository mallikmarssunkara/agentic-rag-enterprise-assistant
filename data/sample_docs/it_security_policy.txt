=========================================
ACME ENTERPRISE - INFORMATION TECHNOLOGY SECURITY POLICY
=========================================
Document ID: ACME-IT-SEC-2026
Version: 5.1
Effective Date: March 15, 2026
Owner: Information Security (InfoSec) Team

1. PURPOSE & SCOPE
------------------
This policy defines the security standards required to protect ACME Enterprise information assets, networks, systems, and data. It applies to all full-time employees, contractors, consultants, and third-party vendors accessing ACME resources.

2. IDENTITY & ACCESS MANAGEMENT
-------------------------------
* Password Complexity: Passwords must be at least twelve (12) characters in length and include at least one (1) uppercase letter, one (1) lowercase letter, one (1) numeric character, and one (1) special character (e.g., !, @, #, $, %, ^, &, *). Passwords must not contain the employee's name, username, or common dictionary words.
* Password Expiration & History: Passwords must be changed every ninety (90) days. The system enforces history validation, preventing the reuse of the last ten (10) passwords. Accounts are locked after five (5) consecutive failed login attempts.
* Multi-Factor Authentication (MFA): MFA is strictly mandatory for accessing all cloud applications, email services, internal portals, and remote access systems.
  - Approved MFA Methods: Microsoft Authenticator app (push notifications or TOTP codes) and hardware security keys (e.g., YubiKey).
  - Restricted Methods: SMS-based MFA is deprecated due to SIM-swapping vulnerabilities and is only permitted under temporary InfoSec exceptions.

3. DEVICE USAGE & WORKPLACE SECURITY
------------------------------------
* Authorized Devices: Only company-issued and managed devices (laptops, desktops, tablets, and smartphones) configured with the ACME Mobile Device Management (MDM) profile may connect to the corporate network or access internal resources (databases, source code repositories, ERP).
* Bring Your Own Device (BYOD): Personal devices may only be used to access corporate email and Slack after installing a specialized MDM container profile. Accessing source code or customer databases from personal devices is strictly prohibited.
* Automatic Screen Lock: Devices must be configured to automatically lock their screens after a maximum of five (5) minutes of inactivity. Employees must manually lock their screens (Win+L / Cmd+Ctrl+Q) when leaving their workstation unattended.
* Anti-Virus & Software Patches: Security patches and antivirus updates are managed centrally by the IT department. Tampering with or disabling security agents (e.g., CrowdStrike, SentinelOne) is a critical policy violation.

4. NETWORK & REMOTE ACCESS SECURITY
-----------------------------------
* Virtual Private Network (VPN): The ACME Secure VPN must be active at all times when accessing corporate systems from public or non-secure networks (e.g., coffee shops, hotels, home Wi-Fi).
* Public Wi-Fi Restrictions: Employees are prohibited from using public, unencrypted Wi-Fi networks without active VPN tunnels. Using cellular hot-spots is preferred for remote work in public locations.
* Software Installation: Employees may only install software from the ACME Software Center. Installation of unapproved third-party software, file-sharing applications, torrent clients, or unsanctioned developer tools requires a signed exception request from the department VP and approval from InfoSec.

5. DATA CLASSIFICATION & SECURITY INCIDENT REPORTING
----------------------------------------------------
* Data Classification: Information is classified into four tiers: Public, Internal, Confidential, and Restricted. Confidential and Restricted data (e.g., customer PII, financial reports, system credentials) must be encrypted both in transit (SSL/TLS) and at rest (AES-256).
* Phishing and Email Security: ACME implements email filtering. Suspected phishing emails must be reported using the "Report Phishing" button in Outlook/Gmail within one (1) hour of detection. Do not forward phishing emails to other colleagues.
* Security Incident Reporting: Any security incident (e.g., lost or stolen laptops, unauthorized access attempts, system breaches, or accidental data leaks) must be reported immediately (within 60 minutes of discovery) via:
  - Security Incident Portal: https://security-portal.acme-enterprise.com
  - Email: security-alerts@acme-enterprise.com
  - InfoSec Hotline: Dial extension 4357 (H-E-L-P) or call +1-888-555-HELP from external phones.
* Policy Violations: Violations of this policy are reviewed by InfoSec and HR. Disciplinary actions may include temporary account suspension, formal warnings, or termination depending on the severity of the infraction.
